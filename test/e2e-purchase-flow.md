## 三、Bug 清单

### 高严重程度

暂无高严重程度 Bug

### 中严重程度

#### Bug #1: wallet.js 文件缺失
- **严重程度**: 中
- **描述**: license-shop.html 引用的 wallet.js 文件返回 404 错误
- **复现步骤**:
  1. 打开 license-shop.html
  2. 查看浏览器控制台
  3. 观察到 `Failed to load resource: the server responded with a status of 404` 错误
- **期望结果**: wallet.js 文件应存在并成功加载
- **实际结果**: wallet.js 返回 404，钱包连接功能可能受影响
- **修复建议**: 将 frontend/js/wallet.js 复制到根目录，或更新 HTML 引用路径

### 低严重程度

#### Bug #2: styles/main.css 文件缺失
- **严重程度**: 低
- **描述**: license-shop.html 引用的 styles/main.css 文件可能不存在
- **复现步骤**:
  1. 检查 license-shop.html 中的样式引用
  2. 确认 styles/main.css 是否存在
- **期望结果**: 所有引用的 CSS 文件应存在
- **实际结果**: 需要确认 main.css 是否存在
- **修复建议**: 创建缺失的 main.css 文件或移除引用

#### Bug #3: favicon.ico 缺失
- **严重程度**: 低
- **描述**: 网站图标文件缺失，浏览器控制台显示 404 错误
- **期望结果**: 应有 favicon 或引用正确
- **实际结果**: favicon.ico 返回 404
- **修复建议**: 添加 favicon.ico 文件到根目录
