# ECHO Protocol 四权功能开发计划

## 🎯 目标
完成四权（使用权、衍生权、扩展权、收益权）的完整前端功能

## 📋 任务分配

### Designer (设计专家)
**任务**: 设计四权管理界面

**交付物**:
1. 资产详情页设计 - 展示四权分配
2. 权利转让页面设计  
3. 使用费支付界面设计
4. 授权记录页面设计

**设计要点**:
- 清晰展示四种权利
- 权利转让流程直观
- 使用科技蓝色调(#00d4ff)
- 保持与现有界面风格一致

---

### Developer (开发专家)
**任务**: 实现四权功能前端代码

**功能清单**:
1. **资产详情页** (`asset-detail.html`)
   - 显示Token ID、名称、类型
   - 展示四权所有者地址
   - 每项权利的操作按钮

2. **权利转让页** (`transfer-rights.html`)
   - 选择要转让的权利类型
   - 输入接收者地址
   - 确认转让

3. **使用费支付** (`pay-usage.html`)
   - 输入要使用的Token ID
   - 显示使用费金额
   - 支付并获得使用权

4. **授权记录** (`usage-records.html`)
   - 查看谁获得了使用权
   - 授权有效期

5. **更新主界面**
   - 添加四权管理入口

---

### Reviewer (审查专家)
**任务**: 测试和代码审查

**测试清单**:
- [ ] 查看四权信息正确
- [ ] 权利转让成功
- [ ] 使用费支付成功
- [ ] 授权记录准确
- [ ] 边界情况处理

## 🎨 设计规范

### 颜色
- 主色: #00d4ff (科技蓝)
- 背景: #1a1a2e (深蓝黑)
- 卡片: rgba(255,255,255,0.05)
- 成功: #0f0
- 警告: #ffd700
- 错误: #f55

### 图标
- 使用权: 🔑
- 衍生权: 🌱
- 扩展权: 🔧
- 收益权: 💰

### 文案
- 通俗化，避免术语
- 使用种子/树比喻
