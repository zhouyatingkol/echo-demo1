## 一、单字命名系统

### 页面主标题 (必须)
每个页面使用一个汉字作为主标识，放在 ECHO 品牌名后。

| 页面功能 | 单字 | 副标题 | 路径 |
|---------|------|--------|------|
| 评论互动 | **评** | 留墨于此，以声会友 | pages/comments-demo.html |
| 试用体验 | **试** | 由此而入，皆可生长 | pages/v4-demo.html |
| 发现探索 | **探** | 寻声而来，觅得所好 | pages/market/discover.html |
| 市场集市 | **集** | 万声汇聚，各取所需 | pages/market/music-market.html |
| 排行榜单 | **榜** | 众声喧哗，自有高下 | pages/market/charts.html |
| 铸造创作 | **铸** | 生此一物，归于你 | pages/creator.html |
| 个人中心 | **境** | 此间天地，皆你所造 | pages/user/dashboard.html |
| 我的资产 | **藏** | 所得所藏，皆是回响 | pages/user/my-assets.html |
| 我的收藏 | **心** | 心有所好，藏之如珍 | pages/user/my-favorites.html |

### 副标题规范
- **长度**: 8-12 字，四字短语 × 2
- **风格**: 诗意、留白、不直白
- **格式**: 前后对仗或递进

**示例**:
- 生此一物，归于你 (铸)
- 寻声而来，觅得所好 (探)
- 留墨于此，以声会友 (评)

## 二、流程步骤命名

多步骤流程使用**单字动词**命名每个步骤。

### 铸造流程 (5步)
```
呈 → 述 → 值 → 权 → 成
```
- **呈**: 上传/呈现作品
- **述**: 描述/命名作品
- **值**: 设定价值
- **权**: 配置权利
- **成**: 完成/铸造

### 命名原则
1. 使用**动词**或**动名词**
2. 每个字必须能独立表达该步骤核心动作
3. 整体读起来有节奏感

## 三、三域分类系统

作品按媒介分为三域，每域配单字 + 八字描述：

| 域 | 单字 | 描述 | 子类 |
|---|------|------|------|
| **音** | 声至心随 | 音乐、音效、声景 |
| **色** | 一色天地 | 图像、影像、视觉 |
| **文** | 字间山河 | 文字、思想、叙事 |

### 描述语规范
- 格式: 四字 + 四字
- 风格: 意境化，不罗列功能
- 避免: "音乐、视频、文章" 这类直白词汇

## 四、四权分离系统

ECHO 核心概念，每个权利配**符号 + 单字 + 解释语**：

| 权利 | 符号 | 单字 | 全称 | 解释语 |
|-----|------|------|------|--------|
| **使用权** | ◈ | 用 | 使用权 | 许他人于特定场景、时限、次数内用之 |
| **衍生权** | ✦ | 衍 | 衍生权 | 许再创作、改编、融合之权 |
| **扩展权** | ◎ | 扩 | 扩展权 | 许跨平台、跨媒体、跨地域之用 |
| **收益权** | ◉ | 益 | 收益权 | 定资产收益之分配则 |

### 符号使用
- **◈** (菱形): 使用权 - 核心基础
- **✦** (四角星): 衍生权 - 创造衍生
- **◎** (双圈): 扩展权 - 向外扩展
- **◉** (靶心): 收益权 - 收益核心

### 解释语规范
- 以"许..."或"定..."开头
- 使用文言句式 (于、之、则)
- 长度控制在 15 字以内

## 五、授权模式命名

### 三合一授权
| 模式 | 单字 | 全称 | 描述 |
|-----|------|------|------|
| **买断制** | 断 | 买断制 | 永久使用 |
| **按次计费** | 次 | 按次计费 | 用几何付几何 |
| **限时授权** | 时 | 限时授权 | 30日起租 |

### 描述语规范
- 单字 + 全称 + · + 简短描述
- 简短描述: 4-6 字，易懂但不俗

## 六、视觉规范

### 颜色系统
```css
/* 宣纸底色 */
--bg-primary: #F8F6F1;

/* 四阶墨色 */
--ink-deep: #1A1A1A;      /* 浓 - 标题 */
--ink-medium: #4A4A4A;    /* 淡 - 正文 */
--ink-light: #7A6E5F;     /* 浅 - 次要 */

/* 点缀色 */
--gold-primary: #B89B6C;   /* 宋金 - 强调、hover */
--seal-red: #A03020;       /* 朱红 - 按钮、印章 */

/* 状态色 */
--realm-sound: #8B9DC3;    /* 音域 - 淡蓝 */
--realm-visual: #C4A574;   /* 色域 - 淡金 */
--realm-text: #7A9E7A;     /* 文域 - 淡绿 */
```

### 字体系统
```css

/* 正文 */

```

### 导航栏规范
- 高度: 60px
- 背景: 透明 → 滚动后 rgba(248, 246, 241, 0.95)
- 品牌: "ECHO · X" (X为页面单字)
- 链接: 单字 "探 集 铸 境"
- 按钮: "入链" (连接钱包)

## 七、文案风格规范

### 语气
- **不**: 科技感、未来感、酷炫
- **要**: 温润、克制、留白、古典诗意

### 句式
- 多用短句
- 适当使用文言词汇 (之、于、则、所)
- 避免网络用语

### 示例对比

| ❌ 现代科技语 | ✅ 东方意境语 |
|------------|-------------|
| 发现音乐 | 寻声而来，觅得所好 |
| 铸造 NFT | 铸声为境，生此一物 |
| 购买 | 缘起 |
| 上传文件 | 呈 |
| 设置价格 | 值 |
| 版权设置 | 权 |
| 确认 | 成 |

## 八、动效规范

### 时间节奏
- 页面切换: 1.2s (如翻书)
- Hover 反馈: 0.3s (如水波)
- 环境动效: 12s 循环 (如呼吸)

### 缓动函数
```css
--ease-ink: cubic-bezier(0.4, 0, 0.2, 1);      /* 渗墨 */
--ease-turn: cubic-bezier(0.25, 0.1, 0.25, 1); /* 翻页 */
--ease-float: cubic-bezier(0.45, 0, 0.55, 1);  /* 悬浮 */
```
