**页面名称**: 资产详情 (Asset Detail)

**页面类型**: 详情页

## 4. 交互逻辑

### 4.1 页面加载
1. 从 URL 获取 tokenId
2. 请求资产详情数据
3. 加载封面图片
4. 初始化播放器

### 4.2 播放器交互
1. 点击播放: 开始播放预览
2. 拖动进度条: 跳转播放位置
3. 音量控制: 调整音量
4. 播放结束: 自动循环

### 4.3 收藏交互
1. 点击收藏按钮
2. 检查登录状态
3. 切换收藏状态
4. 更新按钮图标

### 4.4 购买流程
1. 点击购买按钮
2. 跳转 license-shop.html
3. 预选中使用权

### 4.5 展开更多
1. 点击展开按钮
2. 展开完整描述
3. 按钮文字变为"收起"

## 5. 响应式适配

### 5.1 桌面端 (>= 1024px)
- 封面与信息左右布局
- 权属与描述左右布局
- 固定底部操作栏

### 5.2 平板端 (768px - 1023px)
- 封面与信息上下布局
- 权属与描述上下布局
- 底部操作栏

### 5.3 移动端 (< 768px)
- 封面全宽
- 信息堆叠显示
- 权属简化显示
- 底部固定操作栏

## 6. 状态设计

### 6.1 加载状态
- 封面: 灰色占位
- 文字: 骨架屏
- 播放器: 禁用状态

### 6.2 错误状态
```
┌─────────────────────────────┐
│      ⚠️                    │
│   作品不存在或已下架         │
│   [返回市场]                │
└─────────────────────────────┘
```

### 6.3 未连接钱包
- 显示"连接钱包后购买"
- 播放器仍可试用
