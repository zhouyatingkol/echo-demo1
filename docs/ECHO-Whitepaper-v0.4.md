## 三、四权力框架

### 3.1 权力即四德

《周易·乾卦》：**元亨利贞**。

| 权力 | 卦德 | 本质 | 八档演化 |
|:----:|:----:|:----:|:---------|
| **用** | 元 | 创始 | 禁→锁→借→阅→采→用→持→归 |
| **扩** | 亨 | 通达 | 独→友→圈→域→网→链→元→道 |
| **衍** | 和 | 生化 | 止→赏→评→引→混→融→造→生 |
| **益** | 贞 | 守恒 | 藏→孤→稀→罕→流→众→源→道 |

**元的创始**：从"禁"（封存）到"归"（归属），是存在方式的展开。  
**亨的通达**：从"独"（自闭）到"道"（无界），是连接范围的扩展。  
**和的生化**：从"止"（禁止）到"生"（化生），是衍生能力的释放。  
**贞的守恒**：从"藏"（封存）到"道"（归道），是代谢模式的循环。

### 3.2 权力依赖

**用为基础**。

```
用 = 禁(0)  →  扩 = 独(0)，衍 = 止(0)，自动锁定
用 > 0      →  扩、衍自由配置
```

没有土壤，阳光、水分、空气都无从谈起。  
这是**权利的真实性**，不是技术限制，是存在论的事实。

### 3.3 权力基因

```
基因码 = (用 << 9) | (扩 << 6) | (衍 << 3) | 益
       = 12bit = 4096 种基础配置

这是资产的**先天之命**，创建时即定，不可更改。

**v0.4 扩展**：

**从四权力到六十四卦**：
```

- 天道在上，阳刚之象

## 五、气数流转

### 5.1 何为气数

气者，能量也。  
数者，度量也。

气数是资产在**此时此刻的能量状态**，随时间流转，不可预测，但有规律。

### 5.2 Logistic 混沌映射

```javascript
// 混沌迭代：x(n+1) = r × x(n) × (1 - x(n))
// 当 r = 3.99 时，系统进入混沌状态

function calculateQi(geneCode, timestamp) {
  const r = 3.99;  // 混沌临界
  let x = (geneCode % 9973) / 9973;  // 初始值
  
  // 迭代次数随时间变化
  const iterations = 50 + (timestamp % 50);
  
  // 蝴蝶效应：微小差异 → 巨大结果
  for (let i = 0; i < iterations; i++) {
    x = r * x * (1 - x);
  }
  
  return x;  // 0-1 之间
}
```

**为何用混沌？**

- **确定性**：同一时刻，同一资产，气数固定
- **敏感性**：基因码差1，气数天差地别
- **不可预测**：长期行为无法推算
- **时变性**：随时间自然流转

这正是**"气"的本质**：有序中的无序，规律中的变化。

### 5.3 气数等级

| 气数 | 等级 | 状态 | 诗意 |
|:----:|:----:|:----:|------|
| 0.00-0.25 | dormant | 沉寂 | 潜龙勿用，静待时机 |
| 0.25-0.50 | gentle | 温润 | 见龙在田，德施普也 |
| 0.50-0.75 | vigorous | 蓬勃 | 终日乾乾，反复道也 |
| 0.75-1.00 | fierce | 凛冽 | 或跃在渊，进无咎也 |

气数不是数值，是**状态的诗意表达**。

## 附录 A：数学证明

### A.1 基因码的唯一性

**12bit 权力基因**：
```
基因码 = (用 << 9) | (扩 << 6) | (衍 << 3) | 益
       = 2^12 = 4096 种基础配置
```

**128bit 完整基因码（v0.4）**：
```
完整基因码 = 权力基因(12bit) + 唯一标识(116bit随机)
           = 2^128 种可能
```

**碰撞概率计算**（生日悖论）：
```
P(碰撞) ≈ 1 - e^(-n(n-1)/(2×2^128))
```

| 资产数量 n | 碰撞概率 | 评估 |
|:----------:|:--------:|:----:|
| 100 | 1.47 × 10^-35 | 安全 |
| 1,000,000 | 1.47 × 10^-29 | 安全 |
| 1,000,000,000,000 | 1.47 × 10^-23 | 安全 |

**结论**：即使 1 万亿个资产，碰撞概率仍趋近于零。

### A.2 卦象生成的完备性

**定理**：四权力配置可唯一确定 64 卦之一。

**证明**：
```
上卦 = 用 mod 8 ∈ [0,7]  (8种可能)
下卦 = 扩 mod 8 ∈ [0,7]  (8种可能)
卦序 = 上卦 × 8 + 下卦 + 1 ∈ [1,64]

映射函数 f: (用, 扩) → 卦序 是满射。
∵ |用 × 扩| = 8 × 8 = 64 = |64卦|
∴ f 是双射，配置与卦象一一对应。
```

### A.3 气数算法的混沌特性

**Logistic 映射**：
```
x(n+1) = r × x(n) × (1 - x(n))
```

**当 r = 3.99 时的特性**：

1. **确定性**：给定初始值 x₀，序列 {xₙ} 完全确定
2. **敏感性**：基因码差 1，气数天差地别（蝴蝶效应）
3. **遍历性**：序列在 [0,1] 区间遍历
4. **非周期性**：序列永不重复

**李雅普诺夫指数**：
```
λ = lim(n→∞) (1/n) Σ ln|r(1-2xᵢ)| > 0
```

系统处于**混沌状态**。
