## 一、核心信念

### 1.1 资产即生命

每个数字资产都有自己的：
- **命**——四权力配置，决定其本质
- **气**——随时间流转的能量状态
- **时**——九变势演化，盛衰有时
- **缘**——与使用者、创作者、衍生者的关系

这不是隐喻，是**可计算的存在方式**。

### 1.2 权利即生态

传统思维：权利是边界，是限制。  
ECHO 思维：**权利是生长的条件**。

如同生命需要土壤、阳光、水分、空气：
- **用**是土壤——扎根之地
- **扩**是阳光——能量来源
- **衍**是水分——流动通道
- **益**是空气——循环代谢

**有边界，才能生长。**

### 1.3 生生不息

资产的理想状态不是"被拥有"，而是**"在生长"**。

- 被使用 → 变势上升
- 被引用 → 气数流转
- 被衍生 → 卦象转化
- 被遗忘 → 归于沉寂

生、长、化、藏，**周而复始**。

## 二、四权力框架

### 2.1 权力即四德

《周易·乾卦》：**元亨利贞**。

| 权力 | 卦德 | 本质 | 八档演化 |
|:----:|:----:|:----:|:---------|
| **用** | 元 | 创始 | 禁→锁→借→阅→采→用→持→归 |
| **扩** | 亨 | 通达 | 独→友→圈→域→网→链→元→道 |
| **化** | 和 | 生化 | 止→赏→评→引→混→融→造→生 |
| **益** | 贞 | 守恒 | 藏→孤→稀→罕→流→众→源→道 |

**元的创始**：从"禁"（封存）到"归"（归属），是存在方式的展开。  
**亨的通达**：从"独"（自闭）到"道"（无界），是连接范围的扩展。  
**和的生化**：从"止"（禁止）到"生"（化生），是衍生能力的释放。  
**贞的守恒**：从"藏"（封存）到"道"（归道），是代谢模式的循环。

### 2.2 权力依赖

**用为基础**。

```
用 = 禁(0)  →  扩 = 独(0)，化 = 止(0)，自动锁定
用 > 0      →  扩、化自由配置
```

没有土壤，阳光、水分、空气都无从谈起。  
这是**权利的真实性**，不是技术限制，是存在论的事实。

### 2.3 权力基因

```
基因码 = (用 << 9) | (扩 << 6) | (化 << 3) | 益
       = 12bit = 4096 种基础配置
```

这是资产的**先天之命**，创建时即定，不可更改。

**v0.3 扩展**：
```
完整基因码 = 权力基因(12bit) + 唯一标识(116bit随机)
           = 128bit 总长度
```

既有 ECHO 的特征（前12bit），又有唯一的身份（后116bit）。

## 三、卦象命运

### 3.1 梅花易数起卦

先天八卦：
```
0坤(地)  1艮(山)  2坎(水)  3巽(风)
4震(雷)  5离(火)  6兑(泽)  7乾(天)
```

**从四权力到六十四卦**：
```
上卦 = 用 mod 8    (天道，创造与传播的合力)
下卦 = 扩 mod 8    (地道，生根与延展的承载)
动爻 = (化+益) mod 6 + 1  (人道，变化之机)

卦序 = 上卦 × 8 + 下卦 + 1  (1-64)
```

**为何是梅花易数？**

不是随意的数学映射，是**三才之道**：
- 天道在上，阳刚之象
- 地道在下，阴柔之基
- 人道在中，变动之机

动爻不是装饰，是**此时此刻的变化趋势**。

### 3.2 变卦

动爻所在之爻，阳变阴，阴变阳，得**变卦**。

```
本卦：当下之态
变卦：未来之势
动爻：变化之机
```

例：
- 本卦"火风鼎"，动第2爻 → 变卦"火山旅"
- 鼎者，鼎新之象；旅者，行旅之时
- 从"鼎新"到"行旅"，是创新之后的传播

**不是预测，是趋势**。

### 3.3 六十四卦的诗意

我们不强加定义，让意义自然涌现。

但每一卦有其**气场**：
- 乾为天：开源共享，万物资始
- 坤为地：私人收藏，厚德载物
- 水火既济：流通平衡，相辅相成
- 火水未济：变化未定，无限可能

卦象不是标签，是**场域**。

## 四、气数流转

### 4.1 何为气数

气者，能量也。  
数者，度量也。

气数是资产在**此时此刻的能量状态**，随时间流转，不可预测，但有规律。

### 4.2 Logistic 混沌映射

```javascript
// 混沌迭代：x(n+1) = r × x(n) × (1 - x(n))
// 当 r = 3.99 时，系统进入混沌状态

function calculateQi(geneCode, timestamp) {
  const r = 3.99;  // 混沌临界
  let x = (geneCode % 9973) / 9973;  // 初始值
  
  // 迭代次数随时间变化
  const iterations = 50 + (timestamp % 50);
  
  // 蝴蝶效应：微小差异 → 巨大结果
  for (let i = 0; i < iterations; i++) {
    x = r * x * (1 - x);
  }
  
  return x;  // 0-1 之间
}
```

**为何用混沌？**

- **确定性**：同一时刻，同一资产，气数固定
- **敏感性**：基因码差1，气数天差地别
- **不可预测**：长期行为无法推算
- **时变性**：随时间自然流转

这正是**"气"的本质**：有序中的无序，规律中的变化。

### 4.3 气数等级

| 气数 | 等级 | 状态 | 诗意 |
|:----:|:----:|:----:|------|
| 0.00-0.25 | dormant | 沉寂 | 潜龙勿用，静待时机 |
| 0.25-0.50 | gentle | 温润 | 见龙在田，德施普也 |
| 0.50-0.75 | vigorous | 蓬勃 | 终日乾乾，反复道也 |
| 0.75-1.00 | fierce | 凛冽 | 或跃在渊，进无咎也 |

气数不是数值，是**状态的诗意表达**。

## 五、九变势

### 5.1 变势即呼吸

资产不是死的，它在**呼吸**。

```
变势范围：0-9（九爻）

1/9: 初九 · 潜龙勿用  → 初生，尚未显露
2/9: 九二 · 见龙在田  → 渐显，开始被看见
3/9: 九三 · 终日乾乾  → 勤勉，持续被使用
4/9: 九四 · 或跃在渊  → 试探，可能跃升
5/9: 九五 · 飞龙在天  → 大成，最佳状态
6/9: 上九 · 亢龙有悔  → 过极，需要收敛
7-9/9: 用九 · 群龙无首 → 归元，完成循环
```

### 5.2 变势的流转

**上升**：被使用、被引用、被讨论、被衍生
**衰减**：闲置3天，变势-1

```
使用 → 变势↑ → 满9 → 爻变 → 卦象转化
```

这是资产的**生命周期**：生、长、盛、衰、藏、再生。

### 5.3 用九·群龙无首

变势满9，触发"用九"。

《周易》："见群龙无首，吉。"

不是混乱，是**各正性命，不相统属**。

资产完成一次生命周期，归于混沌，等待下一次觉醒。

**息不是死，是生的准备。**

这是 ECHO 的底线：
