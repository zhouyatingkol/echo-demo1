# ECHO V2 快速上手指南

## 🎯 一、访问平台

打开浏览器访问：
```
https://zhouyatingkol.github.io/echo-demo1/frontend/
```

---

## 🎨 二、生成 V2 资产（带权属蓝图）

### 步骤 1: 连接钱包
1. 确保已安装 MetaMask
2. 切换到 Qitmeer 主网
3. 点击"连接钱包"

### 步骤 2: 进入 V2 生成
1. 在主界面点击 **"🎨 生成 V2"**
2. 进入生成页面

### 步骤 3: 填写基本信息
- **资产名称**: 给您的资产起个名字
- **描述**: 详细描述资产内容
- **类型**: 选择代码/算法/数据/专利
- **IPFS 地址**: 内容上传后的地址
- **内容哈希**: 文件的 keccak256 哈希

### 步骤 4: 配置权属蓝图

#### 使用权设置
- ✅ 允许商业使用
- ✅ 允许修改
- 💰 使用费: 0.001 MEER/天

#### 衍生权设置
- ✅ 允许创作衍生作品
- 💰 衍生授权费: 0.01 MEER
- 📊 父作品收益分成: 10%

#### 扩展权设置
- ❌ 允许功能扩展（可选）

#### 收益权设置
- 💰 收益分成: 100%
- ✅ 自动分配

### 步骤 5: 确认生成
1. 检查所有信息
2. 点击"确认生成"
3. 在 MetaMask 中确认交易
4. 等待交易确认（约 10-30 秒）

---

## 📜 三、版本控制

### 查看版本历史
1. 点击 **"📜 版本控制"**
2. 输入您的 Token ID
3. 查看所有历史版本

### 发布新版本（仅限创作者）
1. 在版本控制页面输入 Token ID
2. 如果是创作者，会看到"发布新版本"面板
3. 填写：
   - 新 IPFS 地址
   - 新内容哈希
   - 更新原因
4. 点击发布

---

## 🔐 四、内容验证

### 验证文件完整性
1. 点击 **"🔐 内容验证"**
2. 选择"文件验证"或"哈希验证"
3. 上传文件或输入哈希值
4. 输入 Token ID
5. 点击验证

### 结果解读
- ✅ **验证通过**: 文件未被篡改
- ❌ **验证失败**: 文件已修改或不匹配

---

## 🌱 五、创建衍生作品

### 步骤 1: 选择父资产
1. 点击 **"🌱 衍生作品"**
2. 点击 **"➕ 创建衍生作品"**
3. 输入父资产 Token ID
4. 查看授权费信息

### 步骤 2: 支付授权费
1. 确认授权费金额
2. 点击支付
3. 在 MetaMask 中确认

### 步骤 3: 生成衍生作品
1. 填写衍生作品信息
2. 配置自己的权属蓝图
3. 点击生成
4. 等待确认

### 收益说明
- 您的衍生作品产生收益时
- 会自动分 10% 给父资产创作者

## 💡 六、常见问题

### Q: 生成需要多少 Gas？
**A**: 约 0.05-0.1 MEER

### Q: 内容哈希怎么计算？
**A**: 
```bash
# 使用 keccak256
keccak256sum file.txt
```
或在内容验证页面自动计算

### Q: 可以修改已生成的资产吗？
**A**: 可以发布新版本，但不能修改原始版本（防篡改）

### Q: 如何转让权利？
**A**: 在 rights-management.html 中可以转让四权

### Q: 别人如何使用我的资产？
**A**: 
1. 支付使用费获得使用权
2. 支付衍生授权费创建衍生作品
3. 系统自动分配收益
