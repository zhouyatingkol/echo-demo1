# ECHO 资产编码系统技术文档

> **万物有灵，皆可生长。**

## 1. 系统概述

ECHO 资产编码系统是一个**双层生命结构**：

- **先天基因**：由创作者设定的四权力参数决定，代表设计意图
- **后天演化**：由实际使用数据驱动的变爻机制，代表现实生长

这种结构让数字资产既有**确定性**（基因码永不改变），又有**流动性**（卦象随生命历程演化）。

## 3. 卦象计算算法

### 3.1 核心公式

```
上卦索引 = (永 + 阔) mod 8
下卦索引 = (延 + 益) mod 8
卦序     = 上卦索引 × 8 + 下卦索引 + 1  (1-64)
```

### 3.2 八卦映射

| 索引 | 卦名 | 符号 | 自然 | 属性 |
|:---:|------|:--:|------|------|
| 0 | 乾 | ☰ | 天 | 健 |
| 1 | 兑 | ☱ | 泽 | 悦 |
| 2 | 离 | ☲ | 火 | 丽 |
| 3 | 震 | ☳ | 雷 | 动 |
| 4 | 巽 | ☴ | 风 | 入 |
| 5 | 坎 | ☵ | 水 | 陷 |
| 6 | 艮 | ☶ | 山 | 止 |
| 7 | 坤 | ☷ | 地 | 顺 |

### 3.3 计算示例

**示例：限量音乐资产**

参数：永=用(5), 阔=域(3), 延=混(4), 益=稀(2)

```
基因码 = 101 011 100 010 = 2582 (十进制)

上卦 = (5 + 3) mod 8 = 0 = 乾(天)
下卦 = (4 + 2) mod 8 = 6 = 艮(山)

卦序 = 0 × 8 + 6 + 1 = 7
卦名 = 第7卦：山天大畜
```

**大畜卦意**：
> "天在山中，大畜。君子以多识前言往行，以畜其德。"

此器用而域，混于稀，如山涵天，蓄积待发。适合限量发行的音乐作品，在特定平台流通，可混剪但保持稀缺。

## 4. 气数计算原理

### 4.1 算法

```javascript
function calculateQi(geneCode, timestamp) {
  const PHI = 1.618033988749; // 黄金比例
  const seed = geneCode * timestamp * PHI;
  return Math.abs(Math.sin(seed));
}
```

### 4.2 特性

- **混沌性**：对初始条件敏感，微小差异产生巨大变化
- **确定性**：相同输入永远产生相同输出
- **有界性**：结果始终在 [0, 1] 区间

### 4.3 诗意映射

| 气数范围 | 等级 | 意境词（易经原文）|
|:-------:|:----:|------------------|
| 0.8-1.0 | high | "飞龙在天，利见大人" |
| 0.5-0.8 | medium | "君子以厚德载物" |
| 0.2-0.5 | low | "潜龙勿用，阳在下也" |
| 0.0-0.2 | dormant | "归藏，坤厚载物" |

## 5. 12变势机制

### 5.1 设计理念

**12** 取自地支之数，代表一个完整的周期。变势积累体现"**积微成著**"的哲学——质变需要量变的积累。

### 5.2 机制规则

```
指标达标 → 变势 +1
满12点   → 触发变爻 → 清空重置
3天无活动 → 变势 -1（衰减）
```

### 5.3 六指标对应

| 变爻 | 指标 | 阈值 | 检测频率 |
|:---:|------|------|----------|
| 上九 | 创作者活跃度 | 20 | 每日 |
| 九五 | 社区参与度 | 50 | 每日 |
| 九四 | 交易活跃度 | 1000 | 实时 |
| 六三 | 衍生数量 | 10 | 每日 |
| 六二 | 连接广度 | 5 | 每周 |
| 初六 | 使用频次 | 100 | 实时 |

### 5.4 触发概率

```
达标概率 = (实际值 / 阈值) × 气数

if 达标概率 > 0.7:
    积累变势
```

### 5.5 与二十四节气的关联

| 节气 | 变势阶段 | 意象 |
|------|:-------:|------|
| 立春-春分 | 0-3 | 萌发 |
| 清明-小满 | 4-6 | 生长 |
| 芒种-处暑 | 7-9 | 繁茂 |
| 白露-立冬 | 10-12 | 成熟 |
| 小雪-大寒 | 触发变爻 | 转化 |

## 6. API 参考

### 6.1 encodeAsset(yong, kuo, yan, yi)

编码四权力为基因码和卦象。

**参数：**
- `yong` (number): 0-7，存在方式
- `kuo` (number): 0-7，连接范围  
- `yan` (number): 0-7，衍生能力
- `yi` (number): 0-7，代谢模式

**返回值：**
```javascript
{
  geneCode: 2582,           // 十进制基因码
  geneCodeBinary: "101011100010", // 二进制
  params: {                 // 参数详情
    yong: { code: "101", name: "用", desc: "按需" },
    // ...
  },
  trigrams: {
    upper: { name: "乾", symbol: "☰", index: 0 },
    lower: { name: "艮", symbol: "☶", index: 6 }
  },
  hexagram: {
    name: "山天大畜",
    index: 7,
    upperIdx: 0,
    lowerIdx: 6
  }
}
```

### 6.2 calculateQi(geneCode, timestamp)

计算资产气数。

**参数：**
- `geneCode` (number): 基因码
- `timestamp` (number): 时间戳

**返回值：**
- `qi` (number): 0.0-1.0

### 6.3 BianShiSystem 类

12变势系统。

**方法：**

```javascript
const bianShi = new BianShiSystem();

// 检查指标
const result = bianShi.checkIndicators({
  usageCount: 150,
  platformCount: 3,
  derivativeCount: 5,
  tradeVolume: 2000,
  communityScore: 60,
  creatorActivity: 25
}, 0.63);

// 返回值
{
  bianShi: 3,           // 当前变势
  max: 12,              // 最大值
  triggered: false,     // 是否触发变爻
  changingLines: [],    // 触发的变爻
  pendingLines: [...],  // 待触发的变爻
  changeSpeed: 'slow'   // 变化速度
}

// 重置
bianShi.reset();
```

### 6.4 calculateChangedHexagram(mainHexIndex, changingLines)

计算变卦（之卦）。

**参数：**
- `mainHexIndex` (number): 主卦序号（1-64）
- `changingLines` (array): 变爻数组 [1-6]

**返回值：**
```javascript

  original: 7,           // 原卦
  changed: 28,           // 之卦
  changedHexagram: "泽风大过",
  changingLines: 2,
  transition: "渐变"
}
```
