## 二、需要修改的具体位置

### 1. creator.html（优先级：高）

```html
<!-- 修改1: Hero标题 -->
<!-- 当前 -->
<h1 class="creator-title">生</h1>
<!-- 应改为 -->
<h1 class="creator-title">铸</h1>

<!-- 修改2: 步骤指示器最后一步 -->
<!-- 当前 -->
<div class="step-dot step-dot--pending" data-step="5">生</div>
<!-- 应改为 -->
<div class="step-dot step-dot--pending" data-step="5">铸</div>

<!-- 修改3: 作品名称占位符 -->
<!-- 当前 -->
<input type="text" class="form-input" id="workName" placeholder="赋之以名" required>
<!-- 应改为 -->
<input type="text" class="form-input" id="workName" placeholder="命名于此" required>

<!-- 修改4: 描述占位符 -->
<!-- 当前 -->
<textarea class="form-input form-textarea" id="workDesc" placeholder="述其灵韵，传其心境" required></textarea>
<!-- 应改为 -->
<textarea class="form-input form-textarea" id="workDesc" placeholder="述其由来" required></textarea>
```

### 2. index.html（优先级：中）

```html
<!-- 修改: CTA区统一使用「铸」而非「生」 -->
<!-- 当前 -->
<h2 class="cta-title">生此 ECHO</h2>
<a href="creator.html" class="cta-btn cta-btn--primary">生</a>
<!-- 建议改为 -->
<h2 class="cta-title">铸此 ECHO</h2>
<a href="creator.html" class="cta-btn cta-btn--primary">铸</a>
```

**说明**: index.html 首页 CTA 区使用「生此 ECHO」与铸造页面的「铸」存在命名冲突。建议：
- 方案A：首页CTA改为「铸此 ECHO」，统一使用「铸」
- 方案B：首页保持「生此 ECHO」（强调创作生命力），铸造页面保持「铸」（强调铸造动作）

### 3. marketplace.html（优先级：低）

```html
<!-- 修改1: 搜索框占位符 -->
<input type="text" id="searchInput" placeholder="寻觅..." />
<!-- 应改为 -->
<input type="text" id="searchInput" placeholder="寻" />

<!-- 修改2: 重试按钮 -->
<button class="retry-btn" onclick="marketplace.loadAssets()">重试</button>
<!-- 应改为 -->
<button class="retry-btn" onclick="marketplace.loadAssets()">再试</button>
```

## 四、总体评价

### 优点
1. **整体风格统一**: 七页面均贯彻东方极简美学，使用单字导航、宋瓷色彩、宣纸质感
2. **提示文案出色**: 加载、错误、空状态提示均使用诗意文案（墨染中、暂无笔墨等）
3. **核心术语统一**: 「入链」「吾」「铸」等核心按钮文案全站一致

### 问题
1. **creator.html 命名不一致**: 页面标题改为「铸」后，Hero区和步骤指示器未同步更新
2. **占位符文案偏差**: creator页面两处占位符使用描述性文字而非指定文案
3. **index.html CTA用词**: 「生此ECHO」与「铸」概念需明确区分或统一

### 修改优先级
- 🔴 **高**: creator.html Hero区、步骤指示器、占位符（4处）
- 🟡 **中**: index.html CTA区用词统一（2处）
- 🟢 **低**: marketplace.html 搜索框、重试按钮（2处）
