## 🌱 ECHO 核心信念（不可动摇）

### 本质定位
**ECHO 不是 NFT，不是技术，是一个权利协议。**

> NFT 是「所有权代币化」—— 回答「谁拥有」  
> ECHO 是「权利分层框架」—— 回答「如何使用、如何扩展、如何分配」  
> 区块链是工具，权利设计才是核心。可用，可不用。

### 核心隐喻
**ECHO 资产 = 入口（Portal）**
- 不是「资产放在哪个平台」
- 是「资产是根节点，万物从此生长」
- 别人的平台和方法，扩展在 ECHO 资产之上

### 生长的哲学
- 四权力是**扩展权**，不是转让权
- 重点在「生长」而非「交易」
- 万物有灵，皆可生长

### 创作者的底线
- 即使 100% 出售收益权，**署名权永远保留**
- 收益权 = 经济回报（可流转）
- 署名权 = 创作者身份（不可分割、永恒归属）

### 设计原则（与核心信念的连接）
**文案四要：**
1. 要单字 —— 留白，不解释，让意义生长
2. 要留白 —— 不填满，给观者参与的空间
3. 要古源 —— 连接传统，时间即信任
4. 要模糊 —— 不定义边界，让扩展可能发生

**视觉四要：**
1. 要白描 —— 本质呈现，不装饰
2. 要呼吸 —— 空间感，不压迫
3. 要时间（动画≥0.8s）—— 慢，是尊重
4. 要手作 —— 人的痕迹，非工业复制

**交互四要：**
1. 要藏 —— 不显露全部，探索即生长
2. 要缓 —— 不急，让关系建立
3. 要印 —— 留下痕迹，建立连接
4. 要缘 —— 相遇是缘，非强制

**设计哲学总结：**
以上所有原则，都在说同一件事 —— **「生长」**。不控制、不填满、不急切，给灵生长的土壤。

### 四权力概念（v0.1）
**四权力 = 使用权 + 衍生权 + 扩展权 + 收益权**

| 权力 | 核心含义 |
|------|----------|
| 使用权 | 使用、体验、消费作品的权利 |
| 衍生权 | 基于原作创作衍生作品的权利 |
| 扩展权 | 将作品扩展至新场景/媒介/联动的权利 |
| 收益权 | 从作品获得经济回报的权利（可拆分 N 份） |

**层级关系：**
- **使用权是基础** → 衍生权 → 扩展权
- 但 ECHO 的使用权是**可控的使用**，有边界、有规则
- 不是随意访问、随意复制，而是「许可制」的使用

**关键区分：**
- 四权力是**扩展权**，不是转让权
- 重点在「生长」而非「交易」
- 收益权可像股权一样拆分（如 10000 份），每份对应未来收益比例

**创作者的底线：**
- 即使 100% 出售收益权，**署名权永远保留**
- 收益权 = 经济回报，署名权 = 创作者身份（不可分割）

**核心洞察：**
- ECHO资产是一个**入口**
- 别人的平台和方法，扩展在 ECHO资产之上
- 资产本身成为连接外部生态的「根节点」

**ECHO vs NFT：**
- ECHO 不是 NFT —— NFT 是「所有权代币化」，ECHO 是「权利分层框架」
- NFT 回答「谁拥有」，ECHO 回答「如何使用、如何扩展、如何分配收益」
- ECHO 可以用区块链实现，也可以不用 —— 技术是工具，权利设计是核心
- NFT 是技术实现，ECHO 是权利协议
