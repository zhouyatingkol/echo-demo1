### 设计原则
**文案四要：**
1. 要单字
2. 要留白
3. 要古源
4. 要模糊

**视觉四要：**
1. 要白描
2. 要呼吸
3. 要时间（动画≥0.8s）
4. 要手作

**交互四要：**
1. 要藏
2. 要缓
3. 要印
4. 要缘

### 音效系统
- v2.0 使用五音调式（宫商角徵羽）
- 不同页面对应不同调式
- 古琴泛音真实模拟（2x-6x泛音列）
- 山谷混响（2.5秒，干/湿 60/40）
