01

什么是 ECHO

不是 NFT，是活着的权益。

NFT 是静态的收藏，是被封存的标本。
ECHO 是活着的权益，是随使用生长的数字生命。

不是拥有，而是相伴。不是结束，而是开始。

ECHO 是一种全新的数字资产协议，它将「动态生长」引入数字权益领域。每一份 ECHO 都承载着创作者的心血，同时也记录着每一次使用、每一次流转的痕迹。随着时间推移，ECHO 会不断进化，价值也会随之流动。

我们相信，真正的数字资产不应该是一张静态的图片或一段固定的代码，而应该是一个有生命力、能与使用者互动、能随时间增值的数字实体。ECHO，就是数字世界的生命形态。

02

快速开始

只需三步，即可开启你的 ECHO 之旅。

1

连接钱包

使用 MetaMask 或其他兼容钱包连接 ECHO 平台

2

获取 ECHO

在市场上购买心仪的 ECHO 作品，或铸造你自己的

3

开始使用

根据权益配置使用作品，见证 ECHO 的生长

首次使用者建议从市场浏览开始，了解不同类型的 ECHO 作品及其权益配置。每个 ECHO 都有独特的生长轨迹，选择与你共鸣的作品，开启一段数字相伴之旅。

03

铸造指南

创作者可以通过 ECHO 协议铸造自己的数字作品。铸造过程简单直观，同时提供丰富的权益配置选项，让你的创作能够以最适合的方式触达用户。

铸造步骤

1. 准备作品素材（音频、图像、文字等）
2. 配置作品元数据（名称、描述、类型）
3. 设置权益模式（使用权、衍生权、扩展权、收益权）
4. 确认铸造参数并提交
5. 等待区块链确认，作品即刻上线

铸造 ECHO 需要支付少量 Gas 费用。建议使用 Layer 2 网络以降低成本。作品铸造后，创作者可以实时监控作品的使用情况和收益分配。

04

权益说明

ECHO 独创的四权模型，为数字资产的流通与使用提供了完整的解决方案。每一份 ECHO 都可以灵活配置这四种权益，实现真正的按需授权。

◈

使用权

基础使用权限，包括播放、展示、体验等。可配置买断、按时长、按次数等多种模式。

◇

衍生权

基于原作进行二次创作的权利。 remix、采样、改编，让创意在传承中绽放。

◎

扩展权

跨平台、跨场景使用的权利。游戏道具、社交头像、虚拟空间装饰等。

◉

收益权

分享作品后续收益的权利。流转分成、使用分成，创作者持续获益。

四权可以独立配置，也可以组合使用。创作者可以根据自己的商业模式灵活设置，用户也可以根据实际需求选择适合的权益组合。

05

技术架构

ECHO 协议构建于以太坊及兼容 EVM 的区块链网络之上，采用模块化设计，确保系统的可扩展性与安全性。

核心组件

• 智能合约层： ERC-721 扩展标准，支持动态元数据更新
• 预言机网络： 链下数据验证与上链
• 索引服务： 高效查询 ECHO 状态与历史
• 前端 SDK： 开发者友好的集成方案

所有核心合约代码均已开源并通过第三方审计。ECHO 协议遵循去中心化原则，不依赖任何单一节点或服务，确保系统的长期稳定运行。

06

常见问题

NFT 是静态的、不可变的数字资产，一旦铸造便固定不变。ECHO 是动态生长的数字权益，其元数据、价值、权益配置都可以随使用而演变。ECHO 更强调「使用」而非「拥有」，是一种更贴近现实经济模式的数字资产形态。

ECHO 的状态更新由智能合约自动执行，所有规则在铸造时即已确定并公开。关键状态变更通过预言机网络验证后上链，确保数据的可信度。同时，所有历史记录永久保存在区块链上，可供随时审计。

ECHO 支持多种收益模式：首次销售、二次流转分成、按次使用费、订阅收入等。收益通过智能合约实时分配，无需人工干预。创作者可以在个人中心查看详细的收益报表和提现记录。

目前 ECHO 主要支持音乐、视觉艺术、文字三类数字作品。未来将逐步扩展至视频、游戏资产、虚拟地产等更多类型。ECHO 的协议设计具有高度扩展性，可以适配各种数字内容形态。

您可以通过以下方式参与：成为创作者铸造 ECHO 作品、作为用户购买和使用 ECHO、为协议贡献代码、参与社区治理投票、成为节点运营者提供索引服务。ECHO 是一个开放生态，欢迎各种形式的贡献。
