## 3. 组件标注

### 3.1 侧边栏

| 菜单项 | 图标 | 状态 | 链接 |
|--------|------|------|------|
| 仪表盘 | 📊 | 激活 | /creator-dashboard.html |
| 上传作品 | 📤 | - | /mint-music.html |
| 批量铸造 | 📦 | - | /bulk-mint.html |
| 收益管理 | 💰 | - | /creator-revenue.html |
| 作品管理 | ⚙️ | - | /rights-management.html |
| ──────── | - | - | - |
| 返回市场 | ← | - | /music-market.html |

### 3.2 统计卡片

| 指标 | 数值 | 变化 | 颜色 |
|------|------|------|------|
| 总收益 | $1,234 | - | 主色渐变 |
| 本月增长 | +23% | 正 | 绿色 |
| 作品数 | 12 | - | 主色渐变 |
| 粉丝数 | 456 | - | 主色渐变 |

### 3.3 图表组件

**收益趋势图**:
- 类型: 折线图
- X轴: 时间 (天)
- Y轴: 收益 (MEER)
- 时间范围切换: 7天/30天/90天/1年

**授权分布图**:
- 类型: 饼图/环形图
- 数据: 使用权/衍生权/扩展权
- 悬停显示数值

### 3.4 热门作品列表

| 排名 | 作品名 | 收益 | 趋势 |
|------|--------|------|------|
| 🥇 1 | Summer Vibes | 456 MEER | 📈 +12% |
| 🥈 2 | Midnight Dreams | 389 MEER | 📈 +8% |
| 🥉 3 | Urban Flow | 234 MEER | 📉 -3% |

### 3.5 动态列表

| 时间 | 类型 | 内容 |
|------|------|------|
| 刚刚 | 🟢 销售 | 用户购买使用权 |
| 5分钟前 | 🟢 收藏 | 获得新收藏 |
| 1小时前 | 🟡 收益 | 收到衍生收益 |
| 昨天 | 🔵 关注 | 新粉丝关注 |

## 4. 交互逻辑

### 4.1 数据加载
1. 并行请求统计数据
2. 显示骨架屏占位
3. 渐进式渲染图表
4. 实时数据 WebSocket

### 4.2 图表交互
1. 悬停显示数据点详情
2. 点击时间范围切换
3. 图表缩放/平移

### 4.3 快速操作
- 点击跳转到对应页面
- 新窗口打开 (Ctrl+点击)

---

## 5. 响应式适配

### 5.1 桌面端 (>= 1024px)
- 侧边栏固定 240px
- 4列统计卡片
- 双栏图表布局

### 5.2 平板端 (768px - 1023px)
- 侧边栏可折叠
- 2列统计卡片
- 单栏图表布局

### 5.3 移动端 (< 768px)
- 侧边栏变为顶部标签
- 2列统计卡片
- 垂直堆叠布局
