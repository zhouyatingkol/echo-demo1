## 🛠️ 技术要点

### WebGL 安全模式
```javascript
// 所有WebGL代码必须包装
try {
    const renderer = new THREE.WebGLRenderer({...});
    // ...
} catch (e) {
    console.warn('WebGL failed:', e);
    // 显示Emoji回退
}
```

### 按钮事件绑定
```javascript
// 正确的绑定方式
document.getElementById('fuseBtn').addEventListener('click', function() {
    if (poolAssets.length >= 2) performFusion();
});

// 错误的绑定方式（可能在WebGL崩溃后失效）
<button onclick="performFusion()">...</button>
```

### 坐标计算
```javascript
// 使用相对于父元素的坐标
const rect = area.getBoundingClientRect();
const centerX = area.offsetWidth / 2;  // 不是 window.innerWidth
const centerY = area.offsetHeight / 2;
```
