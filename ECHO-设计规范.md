# ECHO 视觉与文案设计规范

## 一、核心隐喻系统

### 自然生长隐喻

| 功能概念 | 自然隐喻 | 设计表现 |
|---------|---------|---------|
| 数字作品 | 种子 | 初始形态，蕴含生命力 |
| 创作者 | 园丁 | 培育、照料、收获 |
| 购买者 | 结缘者 | 与作品建立连接 |
| 上链存储 | 入土扎根 | 永恒、稳固 |
| 作品使用 | 光合作用 | 生长、进化 |
| 收益产生 | 开花结果 | 回报、丰收 |
| 版权流转 | 落叶归根/嫁接 | 传递、延续 |
| 时间积累 | 年轮 | 历史、沉淀 |

## 二、界面元素设计

### 1. 种子形态（作品上传中）

```
视觉特征：
├── 形态：椭圆，微有光泽
├── 颜色：土壤褐 + 新芽绿渐变
├── 质感：半透明，内部隐约可见内容轮廓
├── 动效：轻微脉动，如心跳
└── 文案：「一颗种子」
```

### 2. 萌芽状态（作品刚发布）

```
视觉特征：
├── 形态：两片子叶，刚刚破土
├── 颜色：嫩绿色
├── 质感：半透明，带露珠
├── 动效：轻微摇摆，如遇微风
└── 文案：「新芽初绽」
```

### 3. 成长状态（作品被使用）

```
视觉特征：
├── 形态：小树，有枝干和叶片
├── 颜色：深绿色，随使用增多而茂盛
├── 质感：实体感，有纹理
├── 动效：叶片闪烁表示正在使用
└── 文案：「茁壮成长」
```

### 4. 成熟状态（高收益作品）

```
视觉特征：
├── 形态：大树，枝繁叶茂，有果实
├── 颜色：深绿 + 金黄果实
├── 质感：厚重，有年轮纹理
├── 动效：果实微光，随风轻摆
└── 文案：「硕果累累」
```

## 三、文案风格指南

### 语气特征

- **含蓄**：不直接说「买」「卖」，而用「结缘」「收获」
- **诗意**：使用自然意象，营造意境
- **留白**：不填满所有信息，给用户想象空间
- **时间感**：强调时间的流动和沉淀

### 文案层级

| 层级 | 用途 | 示例 |
|-----|------|------|
| 标题 | 页面主题 | 「我的园林」「今日收成」 |
| 引导 | 操作提示 | 「请轻轻放下你的种子」 |
| 反馈 | 操作结果 | 「种子已入土」 |
| 说明 | 辅助信息 | 「此处荒地，待君开垦」 |
| 情感 | 氛围营造 | 「春种一粒粟，秋收万颗子」 |

## 四、关键页面文案

### 首页/发现页

```
页面标题：「园林」

副标题：
- 「移步换景，处处有惊喜」
- 「每一颗种子，都藏着一个世界」

空状态：
- 「园中尚无种子，何不从播种开始？」

加载中：
- 「正在打理园林...」
```

### 上传/播种页

```
页面标题：「播种」

引导语：
- 初始：「请轻轻放下你的种子」
- 拖入文件后：「一颗种子已拾起」
- 填写信息时：「为它培土...」

按钮文案：
- 主按钮：「落款」
- 次要按钮：「暂存草稿」

成功反馈：
- 「种子已入土，静待花开」
- 「今日播种 1 颗，园中共有 N 颗种子」
```

### 作品详情页

```
页面标题：作品名称

状态标签：
- 萌芽期：「新芽初绽」
- 成长期：「茁壮成长」
- 成熟期：「硕果累累」

结缘信息：
- 「此作已与 127 人结缘」
- 「第 128 段缘分，等待开启」

按钮文案：
- 「结缘」
- 「珍藏」
- 「分享此缘」
```

### 收益/收获页

```
页面标题：「收成」

统计文案：
- 「本月收成：N 枚果实」
- 「本季收成：N 枚果实」
- 「年成：丰收」

空状态：
- 「种子在土中，静待发芽」
- 「春种之后，是漫长的等待」
- 「继续播种，终会花开满园」

操作按钮：
- 「收获全部」
- 「存入谷仓」
- 「取出享用」
```

### 个人中心

```
页面标题：「我的园林」

统计卡片：
- 「园中种子：N 颗」
- 「已结缘：N 次」
- 「累计收成：N 枚果实」

时间维度：
- 「入园第 N 天」
- 「一年前的今天，你播下了第一颗种子」

分类标签：
- 「全部种子」
- 「正在生长」
- 「已结缘」
- 「我的珍藏」
```

## 五、动效设计规范

### 时间节奏

```
基本原则：缓入缓出，如自然运动

快速反馈：150ms
├── 按钮点击
├── 状态切换
└── 微交互

标准过渡：300ms
├── 页面切换
├── 元素出现
└── 内容展开

仪式感动画：800-1200ms
├── 落款印章
├── 种子入土
├── 叶子交接
└── 果实收获

生长动画：根据内容长度
├── 加载状态
└── 长时间过程
```

### 动效曲线

```css
/* 自然缓动 */
--ease-natural: cubic-bezier(0.4, 0.0, 0.2, 1);

/* 弹性收尾 */
--ease-bounce: cubic-bezier(0.34, 1.56, 0.64, 1);

/* 庄重缓慢 */
--ease-solemn: cubic-bezier(0.0, 0.0, 0.2, 1);

/* 轻盈飘动 */
--ease-float: cubic-bezier(0.45, 0.05, 0.55, 0.95);
```

## 六、声音设计规范

### 音效映射

| 场景 | 音效 | 描述 |
|------|------|------|
| 落款 | 金石声 | 印章落下，清脆有力 |
| 播种 | 沙土声 | 种子入土，细微柔和 |
| 结缘 | 风声 | 叶子飘动，轻盈自然 |
| 收获 | 木果声 | 果实入篮，温润满足 |
| 生长 | 水流声 | 时间流逝，细微持续 |
| 成功 | 风铃声 | 空灵悠远 |
| 提醒 | 鸟鸣声 | 轻柔不打扰 |

### 音效原则

- **克制**：音效应轻微，不喧宾夺主
- **可选**：提供关闭音效的选项
- **品质**：使用高品质录音，避免电子感
- **东方**：优先使用传统乐器音色

## 七、图标系统设计

### 图标风格

- **线条**：细腻，有书法笔触感
- **圆角**：柔和，不锐利
- **留白**：图标内部留有呼吸空间
- **隐喻**：使用自然意象

### 核心图标

| 功能 | 图标 | 替代文案 |
|------|------|---------|
| 上传/播种 | 🌱 或 种子图标 | 「播种」 |
| 购买/结缘 | 🍃 或 叶子图标 | 「结缘」 |
| 收益/收获 | 🍎 或 果实图标 | 「收获」 |
| 收藏/珍藏 | 🌸 或 花朵图标 | 「珍藏」 |
| 分享/传薪 | 🔥 或 火焰图标 | 「传薪」 |
| 删除/归土 | 🍂 或 落叶图标 | 「归土」 |
| 时间/年轮 | 🌲 或 年轮图标 | 「岁月」 |
| 钱包/谷仓 | 🏚️ 或 谷仓图标 | 「谷仓」 |

## 八、响应式适配

### 桌面端

```
布局：园林式网格，宽松留白
├── 作品展示：大图为先，信息渐进展开
├── 动画：完整仪式感，细腻丰富
└── 交互：hover 效果丰富
```

### 移动端

```
布局：垂直流，单手操作友好
├── 作品展示：卡片式，滑动浏览
├── 动画：保留核心仪式，简化细节
└── 交互：触控反馈明确
```

### 平板端

```
布局：介于两者之间，适配分屏
├── 作品展示：网格 + 详情分屏
├── 动画：完整体验
└── 交互：支持触控 + 手写笔
```

## 九、无障碍设计

### 视觉

- 色彩对比度符合 WCAG 2.1 AA 标准
- 支持高对比度模式
- 动效可暂停/减少

### 听觉

- 所有音效都有视觉反馈替代
- 支持屏幕阅读器

### 操作

- 支持键盘导航
- 触控目标最小 44x44px
- 操作可撤销
