# ECHO 更新日志

## [v0.2.0] - 2026-03-20

### 🔴 重大修正
- **修正先天八卦数序**：0坤(地，全阴) → 7乾(天，全阳)，符合传统易学
  - 旧：0乾、1兑、2离、3震、4巽、5坎、6艮、7坤（错误）
  - 新：0坤、1艮、2坎、3巽、4震、5离、6兑、7乾（正确）

### ✨ 新增功能
- **权力依赖关系**：当"用"=禁(0)时，"扩"和"衍"自动锁定为0
  - 体现"用为基础"的权利层级
  - 视觉反馈：滑块变灰 + 提示"需先开放使用权"
- **九变势系统**：8变势 → 9变势（对应九爻）
  - 1/9: 潜龙勿用 → 5/9: 飞龙在天 → 9/9: 用九·群龙无首

### 🔧 算法升级
- **气数计算**：sin×黄金比例 → **Logistic 混沌映射**
  - 公式：x(n+1) = 3.99 × x(n) × (1 - x(n))
  - 特性：确定性、敏感性、不可预测性、时变性
  - 符合"气"的概念：有序中的无序，周而复始

### 📝 文案调整
- 四权力标签：永阔延益 → **用扩衍益**
- 融入元亨利贞：元之创始、亨之通达、和之生化、贞之正固
- 变势标题：八变势 → **九变势**

### 📚 文档
- 新增《ECHO 协议设计草案 v0.2》
- 包含专家三方验证报告（易学、密码学、工程）
- 明确路线图：v0.3数学强化 → v1.0主网上线
