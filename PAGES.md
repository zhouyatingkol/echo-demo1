# ECHO 平台页面规划

## 网站结构

```
/
├── index.html          ✅ 首页（已完成）
├── market.html         🔄 市场 — 浏览所有作品
├── work/               
│   └── [id].html      🔄 作品详情页
├── creator/            
│   └── index.html      🔄 创作者中心 — 管理作品
├── mint.html           🔄 铸造 — 创建新 ECHO
├── profile.html        🔄 个人中心 — 我的收藏
└── docs/               
    └── index.html      🔄 文档/白皮书
```

## 各页面优先级

### P0 - 核心流程
1. **market.html** — 发现作品（入口）
2. **work-detail.html** — 作品详情+购买（转化）

### P1 - 创作者
3. **creator/index.html** — 创作者仪表盘
4. **mint.html** — 铸造新作品

### P2 - 用户
5. **profile.html** — 个人中心
6. **docs/index.html** — 帮助文档
