# ECHO Protocol 部署文档

## 📦 项目概述

ECHO Protocol 是一个四权分离的数字资产平台，基于 Qitmeer 区块链构建。

### 核心功能
- **铸造数字种子** - 创建带有四权配置的 ECHO 资产
- **资产融合** - 多颗种子融合成树
- **四权管理** - 使用权、衍生权、扩展权、收益权独立流转
- **收益分配** - 树产生的收益按权重分配

## 📋 使用指南

### 1. 准备工作

**安装 MetaMask**
- 浏览器插件商店搜索 MetaMask
- 创建或导入钱包

**添加 Qitmeer 网络**
```
网络名称: Qitmeer Mainnet
RPC URL: https://qng.rpc.qitmeer.io
链 ID: 813
货币符号: MEER
区块浏览器: https://qng.qitmeer.io
```

**获取 MEER**
- 从交易所购买 MEER
- 提现到您的钱包地址

### 2. 铸造数字种子

1. 打开铸造页面
2. 连接 MetaMask 钱包
3. 填写资产信息
   - 名称：如"智能合约算法V1"
   - 描述：资产详细说明
   - 类型：代码/算法/数据/专利
4. 配置四权所有者（默认全部为自己）
5. 点击铸造，确认交易

**Gas费用**：约 0.02 MEER

### 3. 资产融合

1. 确保拥有至少2颗种子
2. 打开融合页面
3. 选择要融合的种子（2-10颗）
4. 设置树的名称和描述
5. 确认融合

**Gas费用**：约 0.02 MEER

### 4. 四权管理

**查看权利**
- 打开四权管理页面
- 查看每项权利的所有者

**转让权利**
1. 选择要转让的资产
2. 选择权利类型
3. 输入接收者地址
4. 确认转让

**支付使用费**
1. 选择要使用的资产
2. 输入使用天数
3. 支付使用费
4. 获得临时使用权

### 5. 收益管理

**分配收益**
1. 打开"我的树"页面
2. 输入要分配的 MEER 金额
3. 点击分配
4. 收益将按权重自动分配

**领取收益**
1. 查看待领取收益
2. 点击领取按钮
3. 确认交易
4. MEER 到账

## 📊 核心概念

### 四权分离

| 权利 | 说明 | 操作 |
|------|------|------|
| **使用权** | 使用资产的权利 | 可转让，可授权他人使用 |
| **衍生权** | 基于资产创作新作品 | 可转让，控制衍生作品 |
| **扩展权** | 扩展资产功能 | 可转让，控制功能扩展 |
| **收益权** | 获得资产收益 | 可转让，获得收益分成 |

### 资产类型

- **种子** - 单个数字资产（代码、算法、数据、专利）
- **树** - 多个种子融合后的资产
- **果实** - 树产生的收益

### 经济模型

- **使用费** - 使用资产需支付费用给收益权所有者
- **收益分成** - 树的收益按种子权重分配
- **权利流转** - 各项权利可独立买卖
