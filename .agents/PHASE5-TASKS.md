# Phase 5 开发任务 - 关注系统 + 批量铸造

## 任务 A: 关注系统 + 消息通知

### A1. 关注系统
**文件**: `frontend/js/follow-system.js`

**功能**:
- 关注/取消关注创作者
- 粉丝列表
- 我的关注列表
- 新作品提醒

**存储**: localStorage + 可选链上存储关注关系

### A2. 消息通知
**文件**: `frontend/js/notifications.js`

**通知类型**:
- 收益到账通知
- 新粉丝通知
- 评论回复通知
- 新作品提醒（关注的创作者）

**UI**: 右上角通知铃铛，下拉通知列表

---

## 任务 B: 批量铸造

### B1. 批量铸造页面
**文件**: `frontend/bulk-mint.html`

**功能**:
- 多文件上传
- 批量填写元数据
- 批量设置授权参数
- 一键铸造多首音乐

**流程**:
1. 上传多个音频文件
2. 批量编辑（可单独修改每首信息）
3. 统一设置授权参数
4. 批量铸造（连续发送交易）
