# 2025-07-08 - ECHO 设计优化

## 集页面整体打磨

### 用户反馈
用户审阅了集页面的4个截图，提出：
1. 第一境排列杂乱，按钮突兀（吾作、随机、价筛选）
2. 卡片展示6个，4+2排列不平衡
3. 第三境没什么意见，但可优化
4. 第四境（观）上面已有色彩，不应再大面积用色

核心要求：东方美学、古朴典雅、克制留白

### 优化方案
创建了 `docs/design/marketplace_polish_plan.md` 详细规划

### 已实施改动

**文案优化**（更有韵味）：
- 遇境：「不期而遇，不约而同」
- 寻境：「循声入色，由色入文」  
- 观境：「声动色起，文以载道」

**布局优化**（删繁就简）：
- 移除"吾作"按钮 → 统一走右下角"铸"入口
- 移除"随机"按钮
- 移除"价"下拉筛选
- 卡片网格改为3列规整布局
- 移除卡片旋转效果

**视觉优化**：
- 寻境：三色块 → 线框卡片，hover才淡淡晕染
- 观境：彩色气泡 → 水墨深浅层次（四阶墨色）

### 待决策：筛选方式
用户认为标签筛选太现代化，讨论了4个方向：
- A. 市招木牌（悬挂式）
- B. 卷轴展开
- C. 集市巷道（拱形门洞）
- D. 极简墨字（推荐）

推荐方向D：最克制、最东方，像古籍批注
