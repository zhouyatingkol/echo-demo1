## 🚀 如何使用

### 1. 访问 Demo
打开 https://zhouyatingkol.github.io/echo-demo1/frontend/

### 2. 连接钱包
- 安装 MetaMask
- 切换到 Qitmeer 主网
- 点击"连接钱包"

### 3. 使用 V2 功能
在主界面点击 V2 功能卡片：
- 🎨 生成 V2 - 创建带权属蓝图的资产
- 📜 版本控制 - 查看/更新资产版本
- 🔐 内容验证 - 验证文件完整性
- 🌱 衍生作品 - 创建和管理衍生作品

### 4. 浏览器测试
在浏览器控制台输入：
```javascript
await V2Test.runAllTests(1)
```
