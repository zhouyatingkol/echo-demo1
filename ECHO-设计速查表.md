# ECHO 设计速查表

## 核心原则

> **「生长」**——数字资产随使用流转而持续进化
> 
> **设计信条**：渐进、含蓄、意在言外、余韵悠长

## 关键场景速查

### 🌱 播种仪式（上传）

```
引导语 → 「请轻轻放下你的种子」
按钮文案 → 「落款」
完成反馈 → 「种子已入土，静待花开」
核心动效 → 印章落下 + 种子入土
```

### 🍃 结缘体验（购买）

```
引导语 → 「此作已与 N 人结缘」
按钮文案 → 「结缘」
完成反馈 → 「你们已结缘」
核心动效 → 两片叶子相触 + 根系相连
```

### 🍎 收获仪式（提现）

```
引导语 → 「园中有果实成熟」
按钮文案 → 「收获」
完成反馈 → 「今日收获 X 枚果实」
核心动效 → 果实依次落入篮中
```

### 🌿 生长脉动（使用反馈）

```
通知文案 → 「你的《作品名》正在别处生长」
视觉表现 → 作品对应部分发光
数据展示 → 「年轮增加 1 圈」
```

## 东方美学参考

| 概念 | 应用 |
|------|------|
| **一期一会** | 每次结缘都是独特的，强调此刻唯一性 |
| **移步换景** | 浏览体验如入园林，处处有惊喜 |
| **意在笔先** | 创作前的留白，信息填写的仪式感 |
| **留白** | 界面不填满，给用户想象空间 |
| **余韵** | 每个操作后都有延续的反馈 |
| **渐进** | 信息渐进展开，不一次性呈现 |
