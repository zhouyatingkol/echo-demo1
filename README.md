# ECHO - 东方美学数字资产权利协议

> **万物有灵，皆可生长**

ECHO 是一个将东方哲学（易经）与数字资产权利设计融合的创新协议。每个数字资产都有独特的"生辰八字"——四权力配置决定基因码，基因码决定卦象命运，时间流转带来气数变化。

---

## 🌟 核心特性

### 四权力框架
| 权力 | 含义 | 元亨利贞 | 范围 |
|:----:|------|:--------:|:----:|
| **用** | 存在方式 | 元之创始 | 禁→锁→借→阅→采→用→持→归 |
| **扩** | 连接范围 | 亨之通达 | 独→友→圈→域→网→链→元→道 |
| **衍** | 衍生能力 | 和之生化 | 止→赏→评→引→混→融→造→生 |
| **益** | 代谢模式 | 贞之正固 | 藏→孤→稀→罕→流→众→源→道 |

**权力依赖**：用=禁时，扩、衍自动锁定——无根之木，无从生长。

### 卦象命运
- **基因码**：12bit = (用<<9) | (扩<<6) | (衍<<3) | 益
- **上卦（天道）**：(用 + 扩) mod 8
- **下卦（地道）**：(衍 + 益) mod 8
- **卦序**：1-64，对应64种生命状态

先天八卦：0坤→1艮→2坎→3巽→4震→5离→6兑→7乾

### 气数流转
使用 **Logistic 混沌映射**计算气数：
```
x(n+1) = 3.99 × x(n) × (1 - x(n))
```
- **确定性**：同一时刻，同一资产，气数固定
- **敏感性**：基因码差1，气数天差地别（蝴蝶效应）
- **时变性**：随时间流转，自然变化
- **不可预测**：长期行为无法推算

气数等级：沉寂 → 温润 → 蓬勃 → 凛冽

### 九变势
```
1/9: 初九·潜龙勿用    5/9: 九五·飞龙在天
2/9: 九二·见龙在田    6/9: 上九·亢龙有悔
3/9: 九三·终日乾乾    7-9/9: 用九·群龙无首
4/9: 九四·或跃在渊
```

资产被使用 → 变势上升 → 满9触发爻变 → 卦象可能转化
