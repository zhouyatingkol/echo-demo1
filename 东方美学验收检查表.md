# ECHO 东方美学验收检查表 V1.0

> **验收理念**: 气韵生动 · 克制留白 · 过目难忘
> **验收原则**: 不说满、不填满、不抢镜

---

## 📋 检查表使用说明

- **通用项**: 所有页面必须检查
- **专项**: 按页面功能特性检查
- **评分**: ✅通过 / ⚠️建议优化 / ❌不符合
- **优先级**: P0(必须) / P1(重要) / P2(加分)

## 一、通用检查项（所有页面）

### 1.1 文案标准

| 检查项 | 标准 | 优先级 | 说明 |
|--------|------|--------|------|
| 导航命名 | 单字命名 | P0 | 境/集/铸/典/域/吾/器，不用双字词 |
| 页面标题 | 单字为主 | P0 | 如「铸」「集」「吾」|
| 副文案长度 | ≤8字 | P1 | 短句留白，不填满 |
| 按钮文案 | 动词优先 | P1 | 「落款」「结缘」「珍藏」而非「确认」「购买」 |
| 空状态文案 | 诗意表达 | P1 | 「此处尚无一物」「待君开垦」而非「暂无数据」 |
| 加载文案 | 自然意象 | P2 | 「正在打理园林」「种子入土中」而非「加载中」 |
| 错误文案 | 含蓄委婉 | P1 | 「缘分未到」「时机未至」而非「操作失败」 |

### 1.2 视觉标准

| 检查项 | 标准 | 优先级 | 说明 |
|--------|------|--------|------|
| 宣纸底色 | #F5F2EA | P0 | 陈年宣纸，禁用纯白 |
| 墨阶层次 | 四阶分明 | P0 | 浓墨(#2A2825)、中墨、淡墨、极淡 |
| 古金点缀 | #A68B5B | P1 | 氧化铜绿，禁用亮金 |
| 白描线条 | SVG手绘感 | P1 | 有笔触感，不用几何图形 |
| 留白比例 | ≥50% | P0 | 上下留白≥8vh，左右≥10vw |
| 字间距 | 0.28em(标题) | P1 | 大字要呼吸 |
| 行高 | ≥1.8 | P1 | 正文舒朗 |
| 圆角 | ≤2px | P1 | 宋瓷微圆角，近乎直角 |

### 1.3 交互标准

| 检查项 | 标准 | 优先级 | 说明 |
|--------|------|--------|------|
| 过渡时长 | ≥0.8s | P1 | 缓入缓出，不急促 |
| 动效曲线 | 墨晕/水波/呼吸 | P1 | `cubic-bezier(0.25, 0.1, 0.25, 1)` |
| 印章按钮 | 朱红/方形 | P1 | 代替普通按钮，可旋转-2deg |
| Hover效果 | 水波纹/墨晕 | P2 | 不突兀，如墨入水 |
| 页面切换 | 翻书/淡入 | P2 | 有仪式感 |

### 1.4 字体标准

| 检查项 | 标准 | 优先级 | 说明 |
|--------|------|--------|------|
| 大标题 | 霞鹜文楷 | P0 | `--font-display` |
| 正文 | 思源宋体 | P0 | `--font-serif` |
| UI元素 | 思源黑体 | P1 | `--font-sans` |
| 英文混排 | 降字号20% | P2 | 英文不突兀 |

## 十、常见违规清单

### ❌ 绝对禁止

- [ ] 使用「购买」「出售」「交易」「订单」等商业词汇
- [ ] 使用「用户」「客户」「消费者」等冰冷称呼
- [ ] 使用红色(#FF0000)等高饱和度颜色
- [ ] 使用几何图形图标（圆形、方形、三角形）
- [ ] 使用纯黑(#000000)和纯白(#FFFFFF)
- [ ] 页面留白<40%
- [ ] 按钮使用直角或圆角>4px
- [ ] 动效时间<0.3s或>3s

### ⚠️ 建议避免

- [ ] 双字以上的页面标题
- [ ] 超过10字的文案
- [ ] 过多的信息密度
- [ ] 闪烁、弹跳等夸张动效
- [ ] 阴影过重（opacity>0.2）
- [ ] 边框过粗（>1px）

## 十一、验收流程

```
1. 阅读页面源码 → 标记文案问题
2. 截图/预览页面 → 标记视觉问题
3. 操作交互流程 → 标记交互问题
4. 填写评分表 → 给出等级
5. 输出整改清单 → 按优先级排序
```
