## 1. index.html - 首页

### 1.1 首字动画 ✓
- **预期行为**：标题"动态数字生命"中"动"字以金色 (#c9a86c) 高亮显示，字号为其他字的 1.4 倍
- **实现方式**：JavaScript 动态生成标题，每个字符使用 `<span>` 包裹，通过 `animationDelay` 实现依次淡入效果
- **动画时长**：首字立即显示，其余字符间隔 0.08s 依次出现
- **状态**：正常

### 1.2 导航链接 ✓
- **ECHO Logo**：链接到 `index.html`
- **市场**：链接到 `market.html`
- **创作**：链接到 `#`（占位）
- **文档**：链接到 `#`（占位）
- **hover 效果**：底部金色下划线从左向右展开，字间距增加
- **状态**：可点击，但创作和文档为占位链接

### 1.3 滚动动画 ✓
- **实现方式**：使用 `IntersectionObserver` API 监听元素进入视口
- **动画效果**：`.fade-up` 类元素从下方 30px 淡入，使用 cubic-bezier(0.16, 1, 0.3, 1) 缓动
- **触发阈值**：元素进入视口 10% 时触发
- **受影响元素**：section-label, section-title, poem-block, feature-item, stat-circle, work-row
- **状态**：正常

### 1.4 其他交互 ✓
- **进入按钮**：点击后平滑滚动到 #about 区域
- **特性卡片悬停**：左侧显示"动/权/实"标记，背景渐变，标题变金色
- **作品行悬停**：左移 1.5rem，名称变金色，字间距增加
- **统计圆圈悬停**：边框变深，轻微放大
