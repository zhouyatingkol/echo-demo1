# ECHO 首页三生卡片白描化改造说明

## 改动文件
`/root/.openclaw/workspace/echo-demo/pages/index.html`

## 改动内容

### 1. SVG图标改造（参考 explore.html 白描风格）

#### 律域 - 古琴白描
- **viewBox**: 改为 `0 0 100 133`（3:4竖卷比例）
- **新增元素**: 
  - 远山淡影（背景层次）
  - 古琴轮廓（白描线条）
  - 七弦（五根琴弦）
  - 琴徽（三个音位标记）
  - 流水波纹（底部装饰）

#### 彩域 - 梅花白描
- **viewBox**: 改为 `0 0 100 133`（3:4竖卷比例）
- **新增元素**:
  - 老干（主枝干）
  - 分枝（两处分叉）
  - 四朵梅花（盛开、半开、含苞、远景）
  - 苔点（底部点缀）

#### 章域 - 书卷白描
- **viewBox**: 改为 `0 0 100 133`（3:4竖卷比例）
- **新增元素**:
  - 书案（底部线条）
  - 双卷轴（左右展开的书卷）
  - 卷轴纹理（简笔文字线条）
  - 轴头（顶部装饰）
  - 镇纸（中间长方形）
  - 毛笔（右侧书写工具）

### 2. CSS样式更新

#### 图标容器
```css
.feature-icon {
    width: 100px;
    height: 133px;  /* 3:4 比例 */
    transition: transform 0.8s ease;
}
```

#### 竖卷效果
```css
.feature-icon--scroll {
    aspect-ratio: 3/4;
    background: linear-gradient(180deg, ...); /* 宣纸质感渐变 */
    border: 0.5px solid rgba(255,255,255,0.1); /* 淡墨边框 */
    padding: 8px;
}
```

#### Hover动画
- 卡片hover时图标 `scale(1.05)`
- SVG透明度从 0.85 过渡到 1
- 过渡时间 0.6-0.8s ease

### 3. 响应式保持
- 保留原有 `.features-grid` 响应式布局
- 移动端自动切换为单列显示
- 竖卷比例在各尺寸下保持一致

## 设计特点

1. **白描风格**: 采用单线勾勒，不填充色彩，体现传统工笔白描韵味
2. **3:4竖卷**: 模拟中国传统书画立轴的比例
3. **层次细节**: 每个图标包含丰富的传统元素细节
4. **宣纸质感**: 通过渐变和边框模拟古画载体
5. **悬停反馈**: 微妙的缩放和透明度变化增强交互感
